# DT Tech — hjemmeside

Landsdækkende freelance inden for tele- og netværksindustrien. Base i Fredericia.

Statisk one-page website (ingen build nødvendig) med lyst nordisk "code"-udtryk, brandfarven Signal Blue og animerede netværksgrafikker.

## Indhold

- `index.html` — hele siden (HTML, CSS og JS i én fil)
- `assets/` — logopakke (DT Tech-logoer, mark, favicons, app-ikoner) samt billeder

## Sektioner

Hero, ydelser, cases, dækning (interaktivt Danmarkskort), proces, om (med "vi ansætter"-note), kontakt og en lille fødselsdags-easter-egg nederst.

## Sådan ser du siden

Åbn `index.html` i en browser (dobbeltklik). Ingen server eller build kræves.

For at få heltebilledet ind: læg et foto i `assets/` som `hero-install.jpg` (valgfrit — ellers vises den animerede baggrund).

## Teknik

- Ren HTML/CSS/JS — ingen frameworks eller afhængigheder
- Fonte via Google Fonts (Space Grotesk, IBM Plex Sans/Mono)
- Responsivt: desktop, laptop/tablet (2 kolonner) og telefon (1 kolonne)
- Respekterer `prefers-reduced-motion`
